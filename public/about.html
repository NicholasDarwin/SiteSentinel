About SiteSentinel

SiteSentinel is a modular, extensible web analysis platform built to help teams find and fix real-world vulnerabilities quickly. It combines deterministic checks with optional AI-driven insights.

Tech

  • Node.js + Express backend
  • Checks under `src/checks/` for easy extension
  • Responsive frontend dashboard
